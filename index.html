Employee Management App

This application is a display of my Full Stack Development skills and it is a basic version of employee management software that could be used in HR department.

It allows user to create/edit/delete data that automatically updates in a SQL Database. There is also an option to view the main Employee table inside Windows Forms that updates all records with a press of Refresh Button and allows the user to search any specific phrases inside the table.

Design has been focused on minimalism and readability as I believe this type of app should focus on simplicity and performance rather than graphics and animations that will slow the program and reduce efficiency of person using it.
